# actividad4_sistemasysenales
Este es un programa de modulación AM (Amplitud Modulada) en Python. 
# Características del Programa
- Generación de señales: Diferentes tipos (senoidal, compuesta, cuadrada, triangular, audio simulado)

-Modulación AM completa: Con control de índice de modulación

- Visualización exhaustiva:

Dominio del tiempo y frecuencia

Espectros de todas las señales

Comparaciones antes/después

-Análisis de ruido:

Diferentes niveles de SNR

Métricas de calidad (MSE, SNR real, correlación)

-Escenarios de distorsión:

Atenuación

Recorte (clipping)

Interferencia

Distorsión no lineal

✅ Análisis de sobremodulación: Efectos de índices > 1
